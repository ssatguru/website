    • SharpDevelop (also referred to as #develop) is a free, open-source (GPL) IDE for C# and VB.NET projects on Microsoft's .NET platform

    The AddInScout is a SharpDevelop addin/plugin, written in C#, which provides helpful information on other AddIns loaded in by SharpDevelop.

    Please note that the AddInScout is now included as part of SharpDevelop. There is no need to install it separately.
